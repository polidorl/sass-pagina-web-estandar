Construyendo sitios web con Sass
 
 Este proyecto presenta el diseño de una página web estándar 
 como ejemplo práctico para demostrar técnicas de programación en Sass. 

 Sass es un preprocesador de CSS que amplía las capacidades de este 
 lenguaje incorporando funcionalidades como el uso de variables, anidación, 
 parciales, importaciones, mixins y funciones.

 Estas características permiten escribir código CSS más organizado, mantenible 
 y reutilizable, lo que fomenta una experiencia de desarrollo más eficiente y 
 escalable.
